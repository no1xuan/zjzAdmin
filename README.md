# ⚡️注意事项
1. 建议小程序发布到线上后再部署管理员网页后台。
如果你想本地测试，就打开JAVA后端的application.yml，然后按需操作修改
2. 默认使用小程序第一个注册（id=1）的当作管理员，其它用户均无法登录管理员后台，请熟知
